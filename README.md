# 1 简介

一个简单易用的短信网关服务，提供客户端 SDK ，支持阿里云，腾讯云，亿美短信发送，适用于中小型公司。

功能如下：

- 应用管理
- 渠道管理
- 模版管理
- 单发短信

# 2 工作原理

![](doc/images/smsliucheng.png)

核心流程如下：

1. 应用程序使用短信平台 SDK ，前端调用应用服务接口发送短信；
2. 应用服务收到短信请求后，调用 SDK 接口根据模版发送短信；
3. 短信平台服务收到请求，根据路由算法选择配置的渠道（比如阿里云、腾讯云）发送短信；
4. 短信成功发送到用户手机。
